# Vibe Vault - Music Playlist Application

This project is a music playlist application with a frontend and backend.

## Deployment Links

- **Frontend:** [https://vibe-vault-frontend-one.vercel.app](https://vibe-vault-frontend-one.vercel.app)
- **Backend:** [https://vibe-vault-backend-three.vercel.app/pjct](https://vibe-vault-backend-three.vercel.app/pjct)

## Tech Stack

- **Frontend:** React, Vite, Tailwind CSS, Axios
- **Backend:** Express, Node.js, MongoDB, JWT Authentication
- **Media Storage:** Cloudinary
